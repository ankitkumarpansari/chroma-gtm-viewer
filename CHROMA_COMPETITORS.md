# Chroma Competitors

## Vector Databases (Core Competitors)

| Competitor | Type | Description |
|------------|------|-------------|
| **Pinecone** | Managed | Managed vector database service |
| **Weaviate** | Open-source | Open-source vector database |
| **Qdrant** | Open-source | Open-source vector search engine |
| **Milvus / Zilliz** | Open-source / Managed | Open-source vector database (Zilliz is the cloud version) |
| **Vespa** | Open-source | Search and vector database platform |
| **LanceDB** | Embedded | Embedded vector database |
| **Marqo** | Open-source | Tensor search engine |
| **Turbopuffer** | Managed | Vector database |

## Cloud/Enterprise Vector Services

| Competitor | Provider | Description |
|------------|----------|-------------|
| **MongoDB Atlas Vector Search** | MongoDB | Vector search capability in MongoDB |
| **AWS OpenSearch** | Amazon | Amazon's search service with vector capabilities |
| **AWS Bedrock** | Amazon | Amazon's managed AI service with embeddings |

## AI/ML Frameworks (Ecosystem Competitors)

| Competitor | Type | Description |
|------------|------|-------------|
| **LangChain / LangGraph** | Framework | LLM application framework with vector store integrations |
| **LlamaIndex / LlamaCloud** | Framework | Data framework for LLM apps with vector capabilities |
| **Haystack (Deepset)** | Framework | NLP framework with vector storage |
| **Cohere** | Platform | AI platform with embedding and rerank services |
